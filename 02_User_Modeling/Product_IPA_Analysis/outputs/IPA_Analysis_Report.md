# 新版车型特征IPA分析报告

## 分析概述

本报告基于BERT模型对用户评论进行情感分析，采用IPA（重要度-绩效分析）方法，
对 48 个车型在 8 个核心特征维度上的表现进行分析。

### 分析维度

1. **外观设计**: 车辆外观造型、颜值、设计风格相关
2. **内饰质感**: 内饰材质、做工、豪华感、精致度相关
3. **智能配置**: 智能科技、辅助驾驶、车机系统相关
4. **空间实用**: 车内空间、储物、实用性相关
5. **舒适体验**: 乘坐舒适性、静音性、减震效果相关
6. **操控性能**: 驾驶操控、动力性能、驾驶感受相关
7. **续航能耗**: 续航里程、能耗表现、充电相关
8. **价值认知**: 性价比、价格、经济性相关认知

### 方法说明

- **情感分析**: 使用BERT模型对评论进行情感分析，生成0-1的情感分数
- **重要度计算**: 基于用户评论中各特征的提及频率计算
- **绩效分数**: 该特征相关评论的平均情感分数
- **IPA象限**: 根据重要度和绩效的中位数划分四个象限

### 四象限说明

- **Q1 优势保持区**（高重要度，高绩效）：继续保持优势
- **Q2 集中改进区**（高重要度，低绩效）：重点投入资源改进
- **Q3 低优先级区**（低重要度，低绩效）：维持现状
- **Q4 过度投入区**（低重要度，高绩效）：可适当减少投入
