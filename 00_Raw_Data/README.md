# Module 00: Raw Data Acquisition

## Description
This directory contains the raw User-Generated Content (UGC) collected from mainstream automotive vertical platforms (e.g., Dongchedi).

## Dataset Content
* **Volume**: Covers **50 top-selling Electric Vehicle (EV) models** in the Chinese market (2023-2024).
* **Format**: `.csv` files per car model.
* **Fields**: Review text, User ID, Car Model, Date.

## Usage Note
These files serve as the **input** for the SSE Module (`01_SSE_Analysis`).

# 模块 00：原始数据获取

## 说明
本目录包含从主流汽车垂直平台（如懂车帝）采集的原始用户生成内容（UGC）。

## 数据集内容
* **规模**: 覆盖中国市场（2023-2024）销量前 50 的主流电动车型。
* **格式**: 每个车型对应一个 `.csv` 文件。
* **字段**: 评论文本、用户ID、车型、日期。

## 使用注意
这些文件是 SSE 模块 (`01_SSE_Analysis`) 的**输入**数据。
